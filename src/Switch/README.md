# Switch

## How to use

```jsx
import { Switch } from "react-quick-settings";

<Switch title="temperature" onChange={handlerOnChange} />;
```

## Props

| propName | propType | defaultValue | isRequired |
| -------- | -------- | ------------ | ---------- |
| title    | string   |              | no         |
| onChange | func     |              | no         |
