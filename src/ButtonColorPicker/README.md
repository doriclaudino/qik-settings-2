# ButtonColorPicker

## How to use

```jsx
import { ButtonColorPicker } from "react-quick-settings";

<ButtonColorPicker color="red" onChange={handlerOnChange} />;
```

## Props

| propName | propType | defaultValue | isRequired |
| -------- | -------- | ------------ | ---------- |
| title    | string   |              | no         |
| onChange | func     |              | no         |
