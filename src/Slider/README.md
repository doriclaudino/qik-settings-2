# Slider

## How to use

```jsx
import { Slider } from "react-quick-settings";

<Slider title="temperature" onChange={handlerOnChange} />;
```

## Props

| propName | propType | defaultValue | isRequired |
| -------- | -------- | ------------ | ---------- |
| title    | string   |              | no         |
| onChange | func     |              | no         |
